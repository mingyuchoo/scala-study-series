# scala-study-series

## How to create a Scala 3 project

```bash
$ sbt new scala/scala3.g8
$ cd <scala-3-project-template>
```

## How to run <scala-3-project-template> project

```bash
$ cd <scala-3-project-template>
$ sbt
sbt:Scala 3 Project Template> ~run
```
