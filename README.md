# scala-study-series

## How to create a Scala 3 project

```bash
$ sbt new scala/scala3.g8
$ cd <project-name>
```

## How to run <project-name> project

```bash
$ cd <project-name>
$ sbt
sbt:Scala 3 Project Template> ~run
```
